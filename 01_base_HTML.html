Balises textuelles, img, table

<p>, <h>, <ul> et <li>

La balise <p> est la balise la plus courante, c'est la balise pour les paragraphes.

La balise <h{$}> est une balise pour les titres, remplacer { $ } par 1,2, 3 jusqu'a 6, chaque balise à son numero et son importance, un titre <h1> signifie que c'est le titre majeur de la page. <h2> pour les titres secondaires et enfin <h3> pour les titres optionnelles.

En théorie une page web à besoin d'un <h1> uniquement. Le reste des titres peu se faire qu'avec des <h2>. Pour le <h3> se sont les titres optionnelles c'est vraiment en cas de besoin, le reste des <h4 a 6> sont des titres à évité du à leurs tailles de police et leurs utilitées.

<ul> est une balise de block qui ne demande qu'un enfant en particulier, les listes à puces <li>. <ul> peut être aussi <ol>, <ol> donnent un comportement différent de <ul>. Ce sont des balises pratiques pour mettre en relations des liens, faire une navigation, un fil arianne etc.

Il est possible d'imbriqué plusieurs listes a puces les une dans les autres et sa un sens, ça nous permet par exemple de faire des sidebars a plusieurs catégories et sous catégories.

<nav>, <a>, <figure>, <figcaption> et <img>

La balise <a> est une balise pour crée des liens hyperText. Un grand mot pour dire qu'avec ça ont peu liée des pages web avec d'autres pages, mais pas que.

<a href="/"> l'attribut href reclâme l'url de destination.

<a href="/" rel="nofollow"> l'attribut rel reclâme une valeur, soit external pour indiquer une redirection est hors site, soit nofollow qui indique au bot de référencement de ne pas suivre ce lien.

<a href="/" target="_blank"> l'attribut target avec la valeur "_blank" dit au navigateur qu'il doit ouvrir un nouveau onglet pour charger la redirection.

<a href="#unIdSituerSurLaPage"> la valeur avec # permet de faire des ancrages sur la page web. Si vous cliquer sur le lien, cela cherchera automatiquement la balise avec ID = "unIdSituerSurLaPage".

<a download="./public/img/image.jpg"> Cette attribut demande une URL d'une ressource afin de pouvoir la télécharger. Cela fonctionne avec des images comme des fichiers html, SQL (utile pour les imports) aussi.

Si l'on place une balise <img src="./assets/img/noimage.png"> dans une balise <a> cela fait une redirection disponible via l'image.

<nav> est un block pour la navigation, généralement conçue pour les sidebars et les Navbar.

<figure> est un block parent réservé au média, sémantiquement on le relie à une image si on ce soucie respecte les standards W3C.

<figcaption> est une légende réserver à la balise <figure>.

<img> permet d'affichée une image. l'attribut alt est obligatoire que si l'image n'est pas ornementale, de même pour l'attribut title. alt indique un texte alternatif si l'image ne se charge pas. alt comme title sert également de source informative pour les personnes avec un handicapes, enfin alt sert aussi au référencement naturel (SEO).

<table>, <th>, <tr>, <thead>, <tfoot> et <tbody>

<table> permet de créer des tableaux, les tableaux peuvent être scindée avec <thead>, <tfoot> et <tbody> respectivement.

<tr> crée une ligne de tableau.

<td> crée une cellule avec la valeur bold. Tandis que <th> crée une cellule en regular font.
